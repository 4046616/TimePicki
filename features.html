Features

Easy to use
Developer can easy to use timepicker in them website using lightweight timepicki.js and timepicki.css
Access with keys
User also pick time use Arrow key via keyboard, and also move like hour to minute field using Tab key
Set time limit
Developer can set hour or minute limit based them project for example time 01 clock to 05 clock only
Change arrow direction
Developer can set arrow operation direction like increment or decrement time
User can type time
In timepicki user can also type hour or minute via keyboard
Clean UI
Timepicki simple and cleas User interface, so user can easily understand and use it
